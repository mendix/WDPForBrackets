What is a Mendix Custom Widget?

 

What is a Mendix Custom Widget?

An extention for your Mendix app.

An extention to your app with a specific functionality.

A solution for a UX problem.

A solution for if you can’t make the functionality needed for your application with
the standard widgets and logic of the Mendix Platform.

When to use a Mendix Custom Widget?

 

When to use a Mendix Custom Widget?

To enhance the UX (User eXperience) of your Mendix app.

To enhance the UX of your application by providing custom functionality. Examples are:

Example 1: Composing a car.

For an app that composes a car, being able to select color, model and type,
the entire interface will change, including the representation
of the car according to the new settings.

Example 2: A menu as filter.

For an app that needs a menu that acts as a filter.

Example 3: Drag and drop products in a webshop basket.

For an app that needs to drag and drop parts of the user interface. (Used within a modern webshop.)

Are we going to build a Custom Widget?

 

Yes we are going to build a Custom Widget!

Best way to learn is by doing.

We are going to create a Reminder Notes widget.

The Reminder Notes custom widget.

Application users can leave notes.

Extra functionality besides the feedback widget.

Notes need to be editable and draggable.

Only notes created by a user can be edited and dragged within a Mendix page (form) by an application user.

What note is from who?

We need to see what notes are from who.

Show / hide and delete notes.

Nodes need to be able to be hidden and shown and if they are from a certain user that user should be able to delete them.
